# 08 — 前端界面

> 前端是一个 Vue.js 3 单页应用，使用 Vite 构建，整个应用只有一个组件文件 `App.vue`（约 400 行）。

---

## 8.1 技术栈

| 技术 | 用途 |
|------|------|
| Vue 3 (Composition API) | 响应式 UI 框架 |
| TypeScript | 类型安全 |
| Vite | 开发服务器 + 构建 |
| 原生 Fetch API | HTTP 请求（无第三方 HTTP 库） |
| ReadableStream | SSE 流式响应解析 |
| 手写 Markdown 渲染器 | 将研报 Markdown 转为 HTML |

---

## 8.2 入口文件（main.ts）

```typescript
// 文件: front/agent_front/src/main.ts

import { createApp } from 'vue'
import App from './App.vue'

createApp(App).mount('#app')
// 极简入口：创建 Vue 应用 → 挂载到 index.html 的 #app 元素
```

### 8.4.4 Markdown 渲染器

```typescript
const markdownToHtml = (markdown: string): string => {
  // 核心渲染流程（7 步）：

  // ① 代码块保护
  //   ```code``` → @@CODE_BLOCK_0@@
  //   先将代码块替换为占位符，防止内部内容被后续规则破坏

  // ② 逐行解析
  //   # text   → <h1>text</h1>
  //   ## text  → <h2>text</h2>
  //   ### text → <h3>text</h3>
  //   - text   → <ul><li>text</li></ul>（自动开闭列表）
  //   空行     → 关闭列表
  //   其他     → <p>text</p>

  // ③ 内联格式
  //   **text** → <strong>text</strong>（粗体）
  //   *text*   → <em>text</em>（斜体）
  //   `text`   → <code>text</code>（行内代码）

  // ④ 链接
  //   [text](url) → <a href="url" target="_blank">text</a>

  // ⑤ 代码块还原
  //   @@CODE_BLOCK_0@@ → <pre><code>原始代码</code></pre>
}

const escapeHtml = (value: string): string =>
  // 将 < > " ' & 转为 HTML 实体，防止 XSS
```
