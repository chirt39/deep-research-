## 项目文件全索引

```
deep_research/
├── config.json                  ← 运行时配置（API Key、模型、数据库连接）
├── .env.example                 ← 环境变量模板
├── pyproject.toml               ← Python 项目定义
├── requirements.txt             ← Python 依赖清单
│
├── app/
│   ├── __init__.py              ← 使 app 成为 Python 包
│   ├── app_main.py              ← FastAPI 入口，创建应用实例
│   │
│   ├── backend/                 ← Web 后端
│   │   ├── __init__.py
│   │   ├── config/
│   │   │   ├── __init__.py
│   │   │   └── settings.py      ← FastAPI 应用配置（端口、CORS）
│   │   ├── schemas/
│   │   │   ├── __init__.py
│   │   │   ├── health.py        ← 健康检查响应模型
│   │   │   └── research.py      ← 研究请求/响应模型
│   │   ├── router/
│   │   │   ├── __init__.py
│   │   │   ├── health_router.py ← /health 健康检查接口
│   │   │   └── research_router.py ← /api/v1/research/run | /stream
│   │   └── service/
│   │       ├── __init__.py      ← WorkflowService 单例工厂
│   │       └── workflow_service.py ← 工作流封装（同步/流式/事件）
│   │
│   └── mult_agents/             ← 核心引擎
│       ├── __init__.py
│       ├── config.py            ← AppConfig：统一配置加载（DeepSeek + 嵌入 + 基础设施）
│       ├── llm_factory.py       ← LLM 工厂：ChatOpenAI + 多后端嵌入模型
│       ├── prompts.py           ← 9 个智能体的 System Prompt
│       ├── state.py             ← ResearchState：27 个字段的状态定义
│       ├── graph.py             ← LangGraph 工作流编排（9 节点 + 条件路由）
│       ├── nodes.py             ← 节点执行引擎（~800 行，核心逻辑）
│       ├── main.py              ← CLI 入口 + Agent 构建 + 记忆管理初始化
│       ├── tools.py             ← 数据采集工具（网络搜索 + AKShare 金融数据）
│       ├── memory/              ← 记忆系统
│       │   ├── __init__.py
│       │   ├── base.py          ← MemoryEntry / MemoryType 数据类
│       │   ├── short_term.py    ← 短期记忆（内存实现）
│       │   ├── long_term.py     ← 长期记忆（SQLite 实现）
│       │   ├── manager.py       ← 记忆管理器（统一接口 + 多后端）
│       │   └── utils.py         ← 记忆提取/合并工具函数
│       └── rag/                 ← 知识库系统
│           ├── __init__.py
│           ├── core.py          ← RAGSystem：向量检索 + 文档入库
│           └── ingest.py        ← 命令行入库脚本
│
└── front/
    └── agent_front/             ← Vue.js 前端
        ├── package.json         ← 前端依赖（Vue + Vite）
        ├── vite.config.ts       ← Vite 构建配置
        ├── src/
        │   ├── main.ts          ← Vue 应用入口
        │   └── App.vue          ← 单文件组件（聊天界面全部逻辑）
        └── index.html           ← HTML 模板
```
